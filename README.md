# BudgetTracker_78
一个简单的记账系统
> 一个基于 Java 开发的命令行记账工具，专为大学生设计，纯本地。

## 📖 项目简介
本项目是 [开源文化运动] 的期末作业。
针对目前市面上记账软件功能臃肿、广告多的痛点，开发了这款极简工具。
它采用 Java编写，通过本地文件存储数据，保障隐私安全。
